# v1.0.0-rc.23.6 (2025-12-07)

Improved Volume card tooltip UX with clearer, more user-friendly language.

### Changed

- **Volume Card Tooltip Optimization**:
  - Removed technical jargon like "(Strict rule applied)"
  - Shortened all descriptions for better clarity
  - Made language more actionable and user-friendly
  - Simplified technical terms (e.g., "circ supply" → "supply")
  - Improved signal descriptions across all states

### Tooltip Changes

**Before → After:**
- "Both price and volume are down (Strict rule applied)" → "Both declining — downward momentum"
- "Sustained bullish — moving averages aligned — sustained buying pressure" → "Sustained upward momentum confirmed"
- "Strong demand, healthy uptrend" → "Strong buying interest"
- "Distribution phase, selling pressure" → "High selling pressure"
- "High activity, potential breakout" → "Increased activity — watch for direction"
- "Price spike (low volume) — Likely thin liquidity or exchange-limited move" → "Price surge on low liquidity"
- "Uptrend losing momentum" → "Weak momentum — watch closely"
- "Selling pressure with reduced participation" → "Decline on reduced interest"
- "Low interest, sideways market" → "Low activity — sideways trend"
- "Quiet market conditions" → "Quiet market"

### Technical

- All tooltips are now more concise and easier to understand
- Removed unnecessary technical commentary
- Maintained accuracy while improving readability
