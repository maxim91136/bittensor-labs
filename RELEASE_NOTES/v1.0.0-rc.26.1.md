# Release v1.0.0-rc.26.1

**Release Date**: 2025-12-08

## Validators Ranking System

### New Features
- **Position Change Indicators**: Visual ranking changes with ▲ (up), ▼ (down), NEW (new entry)
- **History Comparison**: Fetches previous snapshot to calculate position changes using hotkey as identifier
- **Combined Rank Column**: Rank number + change indicator in single column (e.g., "1 ▲2")

### Styling Improvements
- **Unified Monospace Fonts**: Consistent `Courier New` across entire validators table
- **Header Font Consistency**: All headers now 0.9em with 0.05em letter-spacing (matching Subnets)
- **Color-Matched Headers**: DOM % and NOMS headers now use gray/white to match their data values
- **Bold Headers**: All header columns now properly bold (font-weight: 700)

### Technical Details
- Fetches `/api/top_validators_history?limit=2` for previous ranking data
- Uses `hotkey` as unique identifier for validator comparison
- Green ▲ for rank improvements, Red ▼ for drops, Blue NEW for new entries
- Adjusted column widths: 12% | 26% | 24% | 19% | 19%
