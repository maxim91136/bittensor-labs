# v1.0.0-rc.28.3 - Multi-Asset Comparison & Chart Repositioning

**Release Date:** 2025-12-11

## Highlights

### ETH & SOL Price Comparison
Compare TAO performance against multiple assets:
- **BTC** (orange) - Bitcoin comparison
- **ETH** (gray) - Ethereum comparison
- **SOL** (purple) - Solana comparison

All comparisons use normalized percentage view, making it easy to see relative performance.

### Price Chart Repositioned
- Moved from position #5 to **position #2** (after Market Conditions)
- More prominent placement for the dashboard's most-used feature

### Toggle Button Improvements
- **Distinct colors** with visible borders
- **Light Mode support** with darker ETH color (#555)
- **States persisted** to localStorage
