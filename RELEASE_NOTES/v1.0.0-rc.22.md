# v1.0.0-rc.22 (2025-12-06)

Introducing a subtle Matrix-themed Easter Egg for engaged dashboard users.

### Highlights

- **Easter Egg**: "Wake up, Neo..." Matrix Easter Egg
  - Floating green snippet appears after 15-45 seconds (random timing)
  - Visible for 30 seconds with glow and float animations
  - Click triggers full-screen Morpheus message with Matrix rain effect and typewriter animation
  - Gender-neutral message text tailored to the dashboard context
  - Only appears once per session
  - Targets intensive users who spend time exploring the dashboard

### Technical Details

- Fully self-contained JavaScript implementation with inline CSS animations
- No external dependencies or assets required
- Dismissible via ESC key or click anywhere
- Uses Matrix-style green (#0f0) color scheme with Japanese katakana rain effect
- Typewriter effect with variable timing for natural reading experience

### Notes

Pure frontend enhancement. The Easter Egg is designed to be a delightful surprise for users who engage deeply with the dashboard without being intrusive for quick visitors.
