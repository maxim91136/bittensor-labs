# Release Notes: v1.0.0-rc.10

**Release Date**: November 30, 2025  
**Type**: Release Candidate  
**Focus**: Top 10 Validators by Stake Display

---

## üöÄ Highlights

This release introduces the **Top 10 Validators by Stake** card, completing the dual-display of top network participants. Users can now see both the most active subnets AND the largest validators at a glance.

---

## ‚ú® New Features

### Top 10 Validators Card
- **New Dashboard Card**: "Top 10 Validators by Stake" section below Top Subnets
- **Five-Column Layout**: Rank (#), Validator Name, Stake, Dominance %, Nominators
- **Real-Time Data**: Powered by `/api/top_validators` endpoint with dTao API data
- **Beautiful Design**: Consistent styling with Top Subnets card
- **Light Mode Support**: Full theme compatibility

### Backend Infrastructure
- **`/api/top_validators`**: New API endpoint returning top validators
- **`fetch_top_validators.py`**: Python script fetching from Taostats dTao API
- **GitHub Workflow**: `fetch-top-validators.yml` runs hourly at :29
- **Smart Fallback**: Truncated hotkey shown for unnamed validators (e.g., `5Dd8...rWv`)

---

## üìä Data Fields

| Column | Description |

| Validator | Name or truncated hotkey |
