New dashboard card showing the 10 addresses with the highest total TAO balance, complete with identity resolution for known exchanges and on-chain identities.

## ‚ú® What's New

### Top 10 Wallets Card
- **Five-column layout**: Rank, Identity/Address, Balance, Dominance %, Staked %
- **Smart Identity Resolution**:
  - Exchanges API for known exchanges (Binance, MEXC, Bitget, Kraken, Kucoin, etc.)
  - On-chain Identity API for self-registered wallet names (e.g., Taobridge)
- **Visual hierarchy**: Identity name shown prominently, address below in monospace
- **Full theme support**: Proper colors in both dark and light mode

### Backend Infrastructure
- `fetch_top_wallets.py` - Fetches top accounts from Taostats Account API
- `fetch-top-wallets.yml` - Hourly workflow at :39
- `/api/top_wallets` - Cloudflare Pages function serving wallet data
