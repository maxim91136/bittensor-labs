# Release v1.0.0-rc.26.2

**Release Date**: 2025-12-08

## Wallets Ranking System

### New Features
- **Position Change Indicators**: Visual ranking changes with ▲ (up), ▼ (down), NEW (new entry)
- **History Comparison**: Fetches previous snapshot to calculate position changes using wallet address as identifier
- **Combined Rank Column**: Rank number + change indicator in single column (e.g., "1 ▲2")

### Styling Improvements
- **Unified Monospace Fonts**: Consistent `Courier New` across entire wallets table
- **Header Font Consistency**: All headers now 0.9em with 0.05em letter-spacing (matching Subnets/Validators)
- **Color-Matched Headers**: DOM % and Staked headers now use gray/white to match their data values
- **Bold Headers**: All header columns now properly bold (font-weight: 700)

### Technical Details
- Fetches `/api/top_wallets_history?limit=2` for previous ranking data
- Uses `address` as unique identifier for wallet comparison
- Green ▲ for rank improvements, Red ▼ for drops, Blue NEW for new entries
- Adjusted column widths: 12% | 28% | 24% | 18% | 18%

### Complete Ranking System
All three Top 10 cards now have ranking indicators:
- **Subnets**: Uses `netuid` as identifier
- **Validators**: Uses `hotkey` as identifier
- **Wallets**: Uses `address` as identifier
