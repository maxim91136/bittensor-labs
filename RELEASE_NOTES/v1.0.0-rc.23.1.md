# v1.0.0-rc.23.1 (2025-12-06)

Layout optimization for Fear & Greed Index and Miner Map cards.

### Improved

- **Card Layout**: Fear & Greed Index and Miner Map cards now display side-by-side on wider screens
  - New responsive container (`.fng-map-container`) with CSS Grid
  - Side-by-side layout at ≥800px viewport width
  - Stacked layout below 800px for better mobile experience
  - Equal card widths for balanced visual appearance
  - Maximum container width: 1400px

- **Fear & Greed Card Styling**
  - Centered spoon graphic using `left: 50%` + `translateX(-35%)`
  - Better horizontal positioning for the barometer/tachometer
  - Adjusted Matrix quote "there is no spoon..." positioning
    - Repositioned to `top: 6px, left: 12px`
    - Reduced font size to `0.68rem` for better spacing
    - Lower opacity (`0.45`) for more subtle appearance

### Technical Details

- New CSS Grid container: `.fng-map-container`
  - Mobile (< 800px): `grid-template-columns: 1fr`
  - Desktop (≥ 800px): `grid-template-columns: repeat(2, 1fr)`
- Spoon background positioning: `object-position: center center`
- Matrix quote styling updates for improved header spacing

### Files Changed

- `index.html`: Added `.fng-map-container` wrapper around F&G and Miner Map cards
- `style.css`: New responsive grid layout and spoon centering adjustments
- `VERSION`: Bumped to v1.0.0-rc.23.1

### Notes

Pure frontend layout enhancement. No backend or API changes. Improves visual hierarchy and makes better use of horizontal space on desktop displays.
